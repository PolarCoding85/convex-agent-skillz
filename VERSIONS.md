# Version History

## v1.0.0 — Initial Release

Initial collection of Convex skills for Claude Code.

### Skills
- **convex** — Core backend development (queries, mutations, actions, schemas, scheduling, file storage, search, Next.js)
- **convex-agent** — AI agents with persistent threads, tool calling, and streaming
- **convex-components** — Component patterns (Rate Limiter, Aggregate, Workpool, Workflow, and more)
- **convex-auth** — Universal authentication patterns (provider-agnostic)
- **convex-clerk** — Clerk authentication integration
- **convex-workos** — WorkOS AuthKit integration

### Extras
- **convex-expert** agent — Auto-activates in `convex/` directory
- **/create-features** command — Converts plans into feature documentation
